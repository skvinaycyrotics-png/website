# **App Name**: CyroSecure

## Core Features:

- Service Showcase: Interactive display of services: Infrastructure, Security, Audio-Visual, Smart Building, Software, Renewable Energy, showing key benefits and tech stacks.
- Project Visualizer: Filterable project gallery with detailed case studies and before/after comparisons.
- Expert Match Tool: A tool that matches user inquiries with the appropriate internal expert based on service interest. LLM reasoning helps to interpret vague service interests, ensuring optimized and accurate routing within the company.
- Client Portal: Secure login for clients to track project progress and access support.
- ROI Calculator: Interactive tool to calculate potential return on investment for different technology solutions.
- Careers Page: Dynamic listing of job openings with application submission.
- Contact Form: Integrated contact forms for general, sales, and support inquiries, linked to Firestore database.

## Style Guidelines:

- Primary color: Deep ocean blue (#3478A9) to evoke trust and professionalism.
- Background color: Light grayish-blue (#F0F4F7), maintaining the calm mood.
- Accent color: Bright sky blue (#74B9FF) for CTAs and highlights.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and objective feel.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use clear and professional icons for services and features, aligning with the technology sector.
- Subtle, professional animations on hover and scroll events to enhance user experience.
